Intermediate

Now that we have shown you how to show basic content using bootstrap we are going to add some extra functionality using bootstrap. The next thing that most developers would need to know is how to make content dynmaic. In most apps the way to do this is with user interaction. This page is going to dive into different GUI features that allow users to interact with your web page.


Buttons

The first important way to interact with a gui is through the use of buttons. Bootstrap provides several different button options for us allowing developers to specify the exzact look and feal. Hear are their options:



Each one of these buttons can be used in a different situation providing their desired effect. In order to create these buttons you do the following:


            
<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-link">Link</button>
<button type="button" class="btn btn-primary">Primary</button>     
<button type="button" class="btn btn-info">Info</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-danger">Danger</button>

Button Groups

Bootstrap also provides a way to group buttons together. This can be a line of buttons or a group of buttons. Just like many other aspects of bootstrap, the way to do this is to put the buttons inside of a special container. 

            
<div class="btn-group">
    <button type="button" class="btn btn-primary">Yes</button>
    <button type="button" class="btn btn-primary">Maybe</button>
    <button type="button" class="btn btn-primary">No</button>
</div>

Forms

Buttons are nice and can provide a great user expierence but you can't do anything really useful unless you have user input. User input allows users to log in, proivide information, and interact with the system in a way that you just can't do with anything else. Thy way html does this is with the use of forms. Luckly, for us bootsrap has a way to make forms look nice and professional.


Below is an example of a very standard login form.



            
<form role="form">
  <div class="form-group">
    <label for="email">Email address:</label>
    <input type="email" class="form-control" id="email">
  </div>
  <div class="form-group">
    <label for="pwd">Password:
    <input type="password" class="form-control" id="pwd">
  </div>
  <div class="checkbox">
    <label><input type="checkbox"> Remember me</label>
  </div>
  <button type="submit" class="btn btn-default">Submit</button>
</form>

Inputs
