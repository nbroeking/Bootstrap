Intermediate

Now that we have shown you how to show basic content using bootstrap we are going to add some extra functionality using bootstrap. The next thing that most developers would need to know is how to make content dynamic. In most apps the way to do this is with user interaction. This page is going to dive into different GUI features that allow users to interact with your web page.


Buttons

The first important way to interact with a gui is through the use of buttons. Bootstrap provides several different button options for us allowing developers to specify the exact look and feel. Hear are their options:



Each one of these buttons can be used in a different situation providing their desired effect. In order to create these buttons you do the following:


            
<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-link">Link</button>
<button type="button" class="btn btn-primary">Primary</button>     
<button type="button" class="btn btn-info">Info</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-danger">Danger</button>

Button Groups

Bootstrap also provides a way to group buttons together. This can be a line of buttons or a group of buttons. Just like many other aspects of bootstrap, the way to do this is to put the buttons inside of a special container called a btn-group. 

            
<div class="btn-group">
    <button type="button" class="btn btn-primary">Yes</button>
    <button type="button" class="btn btn-primary">Maybe</button>
    <button type="button" class="btn btn-primary">No</button>
</div>

Forms

Buttons are nice and can provide a great user experience but you can't do anything really useful unless you have user input. User input allows users to log in, provide information, and interact with the system in a way that you just can't do with anything else. Thy way html does this is with the use of forms. Luckey, for us Bootsrap has a way to make forms look nice and professional.


Below is an example of a very standard login form.



            
<form role="form">
  <div class="form-group">
    <label for="email">Email address:</label>
    <input type="email" class="form-control" id="email">
  </div>
  <div class="form-group">
    <label for="pwd">Password:
    <input type="password" class="form-control" id="pwd">
  </div>
  <div class="checkbox">
    <label><input type="checkbox"> Remember me</label>
  </div>
  <button type="submit" class="btn btn-default">Submit</button>
</form>

Inputs

As you can see from above bootstrap provides several types of inputs we showed you a basic text field above using the form-control class. Following the theme of the rest of the tutorial, bootstrap provides more options.


Textareas


The code for this looks like:


<div class="form-group">
    <label for="comment">Bootstrap also provides text areas.</label>
    <textarea class="form-control" rows="5" id="comment"></textarea>
</div>

Check Boxes

In addition to text fields there are also two types of check boxes available for us to use. They are normal and inline check boxes. The main difference between the two is that normal text boxes align vertically and inline textboxes align on the same line.


Normal Checkboxes

            
<div class="checkbox">
  <label><input type="checkbox" value="">Pick Me</label>
</div>
<div class="checkbox">
  <label><input type="checkbox" value="">No Pick Me</label>
</div>
<div class="checkbox disabled">
  <label><input type="checkbox" value="" disabled>How About Me</label>
</div>

Inline Checkboxes

Inline check boxes look very similar to normal checkboxes but instead of appearing vertically they appear horizontally like:



The code for this is very similar the normal checkboxes but the label uses the checkbox-inline class instead of the checkbox class for div.

            
<label class="checkbox-inline"><input type="checkbox" value="">Pick me</label>
<label class="checkbox-inline"><input type="checkbox" value="">No Pick Me</label>
<label class="checkbox-inline"><input type="checkbox" value="">How About Me</label>

Bootstrap Select Lists

Finally, the last form of input that we are going to talk about are bootstrap's select lists. Bootstrap provides two ways to show select lists. They can either be shown in a drop down or in a multiple select list.


Drop Down

The drop down is another fundamental element for gathering user input. It allows users to select an option from a list of choices. The bootstrap drop down looks like:

In order to create a drop down you need to create a form that has a select tag that uses the class .form-control. This looks like:

            
<form role="form">
    <div class="form-group">
        <label for="sel1">Drop Down Pick the One you Want:</label>
        <select class="form-control">
            <option>You can select me</option>
            <option>You can select me too</option>
            <option>I am the best</option>
        </select>
    </div>
</form>

The other kind of Select List is the multiple select list.


The code to implement this:

            
<form role="form">
    <div class="form-group">
        <label for="sel2">This is a multiple select list: You can hold shift to select more than one:</label>
            <select multiple class="form-control">
                <option>Select Me</option>
                <option>Hold Shift to Select me Too</option>
                <option>You can select all of us if you want to</option>
            </select>
        </div>
        </form>
