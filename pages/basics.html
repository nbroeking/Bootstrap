Basics

Now that we have talked about all the prerequisites for bootstrap we can start developing friendly and responsive web pages. Bootstrap provides many different css classes for styling pages but overall it boils down to putting normal html inside of a container. So first we are going to talk about containers.

Containers

The first important thing to know is that bootstrap requires a container for everything. If you don't have everything inside a container then it wont get the correct styling and wont look the way that you intended. With containers bootstrap gives two options.

1.) .container class

The .container class provides a responsive fixed width container

2.) .container-fluid

The .container-fluid class provides a container that spans the full width of the view.

Example

The following example shows the code and the result for a basic webpage using Boostrap containers.

Code

<!DOCTYPE html>  
<html lang="en" >

<head>
    <title>Bootstrap Tutorial</title>
    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
    
    <!-- Latest compiled and minified CSS -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
    
    <!-- Optional theme -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap-theme.min.css">
        
    <!-- Latest compiled and minified JavaScript -->
    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>
<body>
    <div class="container-fluid">
        <h1>Hello World</h1>
        <p>Hello everyone reading this</p>
    </div>
</body>
</html>

This code provides the following result.

Hello World

Hello everyone reading this


Containers are very important with bootstrap because that is the only really way to style groups of tags together. At its essence, html is designed to display information. Bootstrap takes this information groups it together and styles it in a way that makes sense to the users. Bootstrap has plenty of different options for containers and the rest of this section is going to focus on the different ways to group data and display it.


JumboTron

A Jumbotron is another type of container that is used to bring extra attention to some content. They are an easy way to draw attention to part of the page while keeping everything concise. In order to use them you must use the .jumbotron class, like we do below.


<div class="jumbotron">
    <p>This is some attention grabbing text.</p>
</div>

This produces the following effect.

Attention!!!!

This is some attention grabbing text.


Tables

The next type of container to use is a table. Tables are not unique to bootstrap, they are used in html all the time. Bootstrap adds several css classes though that makes bootstrap tables look way better than the html counterpart. To create a basic table in Boostrap you use the <table> tag with the class .table. For example:

Normal Table

Firstname Lastname Email
Nic Broeking nicsEmail@broeking.com
Josh Rahm josh@rahm.com
Matt Burbernak Matt@Bubernak.com

The Code:


<div class="container">
    <h2>Normal Table</h2>         
    <table class="table">
        <thead>
            <tr>
                <th>Firstname</th>
                <th>Lastname</th>
                <th>Email</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Nic</td>
                <td>Broeking</td>
                <td>nic@broeking.com</td>
            </tr>
            <tr>
                <td>Josh</td>
                <td>Rahm</td>
                <td>josh@rahm.com</td>
            </tr>
            <tr>
                <td>Matt</td>
                <td>Burbernak</td>
                <td>Matt@Bubernak.com</td>
            </tr>
        </tbody>
    </table>
</div>

There are plenty of more types of tables that you can use. Bootstrap has popup tables, responsive tables, and even condensed tables. For the purposes of our tutorial we arn't going to show you all of these but you can get a general idea.


Wells

The last container that we are going to show you are wells. Wells are similar to a jumbotron except that they arn't as exaggerated. They are a nice way to group text together into a text box or similar type of container. To create a well all you need is a container that uses the class .well For example


            
<div class="well">This is a basic well.</div>

This looks like:


This is a basic well.

Of course just like the tables. Bootstrap provides several different looks for wells. The options are Small Well, Normal Well, and Large Well


The code and output for each one looks like.


            
<div class="well-sm">This is a small well.</div>

This is a small well.

            
<div class="well-sm">This is a large well.</div>

This is a large well.
