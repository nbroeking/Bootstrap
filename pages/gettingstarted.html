Getting started

Like we mentioned on the previous page Bootstrap is a front end web framework used to display html. This means that everything can be done on the front end. Unlike a lot of other web frameworks, we don't need to modify anything on the server side. We can jump right in to programming with Bootstrap.

Step One: Importing the Bootstrap libraries

There are two ways to go about using the Bootstrap libraries.

1.) Host the Bootstrap Libraries

The first way is to download the Bootstrap libraries from their website and host them on your webserver. Then you can import them in your header. This method however is very rarely used because the alternative is much quicker and more efficient.

2.) Use a CDN

Most people use a CDN or a Content Delivery Network to serve the pages. This is far more efficient because the majority of people do this allowing for much faster load times because your browser will cache the libraries from other pages. In order to do this your header should look something like this.



<!DOCTYPE html>   
<html lang="en" >
<head> 
    <title>Bootstrap Tutorial</title> 
    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
        
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
        
    <!-- Latest compiled and minified CSS -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
            
    <!-- Optional theme -->
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap-theme.min.css">
            
    <!-- Latest compiled and minified JavaScript -->
    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
</head>

This Header downloads the libraries for jQuery, and the different components of bootstrap. Once you have your header were free to start developing with bootstrap.
