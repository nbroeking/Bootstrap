Introduction

Bootstrap is an extremely powerful web framework that lets developers build very professional looking web pages extremely quickly. In this tutorial we are going to teach you the fundimentals of this extremely powerful tool. Before we dive into bootstrap it is important to understand what Boostrap is.


What is Bootstrap?

Bootstrap is a client-side framework. It is primarily a CSS library, although it has been extended to work with JavaScript. Bootstrap defines a set of consistent CSS templates for building large, professional looking web applications. From the beginning, bootstrap was supposed to be quick to develop and easy for both the developer and the end user. One of the biggest wins with bootstrap is its ability to take advantage of the CSS media flags to create consistent looking applications for both desktop and mobile clients.

Bootstrap allows the user to create a single source and Bootstrap will auto-magically dynamically resize to fit a whole array of different browser sizes.

One of the big wins with bootstrap is the ease of being able to create a web application with the ability to dynamically resize to fit on mobile platforms as well as desktop platforms.

Where does it come from?

Bootstrap was developed at Twitter by Mark Otto and Jacob Thornton. They released it as an open source project August 2011. Once it became open sourced it became so popular that it was the #1 project on Github in 2014.

What should I know before looking at this page?
