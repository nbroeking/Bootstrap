Scaffolding

From the very beginning, Bootstrap was designed to work with mobile well with mobile devices as well as desktop devices. To accomplish this, Bootstrap rolled out its own kind of layout system it calls Scaffolding. This allows the developer to layout widgets in the terms of an implicit grid bootstap has that by default has 12 columns. So if I want to make a <div> that fills up the entire container, its as simple as saying <div class="col-md-12">

The code for this is pretty straight-forward, all we have is: 
            <div class="row">
                <div class="col-md-12 example">12</div>
                <div class="col-md-6 example">6</div>
                <div class="col-md-6 example">6</div>
                <div class="col-md-3 example">3</div>
                <div class="col-md-3 example">3</div>
                <div class="col-md-6 example">6</div>
                <div class="col-md-1 example">1</div>
                <div class="col-md-2 example">2</div>
                <div class="col-md-3 example">3</div>
                <div class="col-md-6 example">6</div>
            </div>
This scaffolding is extremely useful when designing web applications for several reasons. First, these elements are responsive. That means that they will automatically shuffle to ensure that the elements will fit on the screen that they are trying to fit on. You'll notice that if you zoom in on your browser or look on a mobile device, this scaffolding will turn into a stack of scaffolds. This is something that Bootstrap does automatically to better fit the content on a smaller device.
