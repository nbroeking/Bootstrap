Introduction

Bootstrap is an extremly powerful web framework that lets developers build extremly professional looking web pages extremly quickly. In this tutorial we are going to teach you the fundimentals of this extremly powerful tool. Before we dive into bootstrap it is important to understand what boostrap is.


What is Bootstrap?

Bootstrap is a front-end framework, that provides html, and css based templats as well as optional javascript plugins. It is designed to give programmers the ability to provide their users a friendly, interactive, and responsive user interface.
