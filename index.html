Introduction

Bootstrap is a powerful front end web development tool that allows for quick development as well as clean and professional looking web pages. Boot strap was developed by Mark Otto and Jacob Thornton at Twitter. and became an open source project on Github in August 2014.



We are going to try and walk you through an introduction to bootstrap using a bootstrap website!!! You can use the navigation bar at the top of the page to cycle through each chapter.
